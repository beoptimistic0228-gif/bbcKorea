# 설교 본문 × 찬송 연결 구조 분석 (155개 주일 전수)

> **결론 먼저:** 목사님의 선곡은 **① 포지션이 '정서 범위'를 강하게 규정**하고(마무리=헌신, 헌금=성령·기도·간증, 오전=경배+인도), 그 안에서 **② 그 주 본문의 '적용 정서'가 구체적 곡을 고른다.** 즉 **본문의 '책·사건'보다 '포지션 고정 성격 × 그 주 적용 정서'의 이중 필터**로 결정된다. 성도의 1인칭 응답(헌신·확신·성결·열망) 곡을 하나님 속성 찬양보다 선호.

- **근거:** `설교_찬송_실적.csv`(**155주일, 2022-06~2026-06**; 2025~2026 로마서·엘리야 강해 포함) + 분류는 `침례교찬송가_가사DB_마스터.md`. 전수 대조표는 `본문_찬송_enriched.md`.
- 아래 %는 각 포지션에서 그 분류가 차지한 비율.
- **⚠ 품질 주의:** 개별 행 일부는 목사님 기입오류가 당일 수정됐으나 DB 미반영된 노이즈일 수 있음 → **집계·패턴은 신뢰, 단일 행 본문↔찬송 연결은 과신 금지.** 2025 일부·2026 2~3월(로마서 14~15 추정) 소수 주일은 원본 미입력/공백.

## 1. 포지션별 '정서 지문'(가장 큰 구조)

### 마무리(파송) — **헌신 58.4% 압도** 🔒
| 분류 | 비율 |
|---|---|
| **헌신** | **58.4%** |
| 초대와 영접 | 8.4% |
| 기도 | 7.8% |
| 열망 | 7.1% |
| 평안과 위로 | 3.9% |

→ 마무리는 **"헌신·결단의 파송"** 자리. 곡 풀도 좁음(402·399·401·396·403 …). 추천 시 **헌신 계열에서 그 주 색채로 선택**.

### 헌금 — **성령·간증·기도·평안**(내면·성찰) 🙏
| 분류 | 비율 |
|---|---|
| 성령 | 15.5% |
| 간증과 찬양 | 12.3% |
| 기도 | 8.4% |
| 평안과 위로 | 7.7% |
| 확신 | 7.1% |

→ 헌금은 **기도·성찰·간증의 내면 정서**. 대표 **179 내 눈을 열어 주소서〔성령〕**(21회)가 사실상 기본값. 추천 시 **179 기본 + 그 주 정서로 교체**(힘=346, 평안=338, 확신=285 …).

### 오전(경배) — **인도와 돌보심 22% + 사랑·경배**(경배로 열고 응답) 🎶
| 분류 | 비율 |
|---|---|
| 하나님의 인도와 돌보심 | 21.3% |
| 하나님의 사랑과 은혜 | 11.3% |
| 경배와 찬양 / 찬양과 경배 | 16.1% |
| 그리스도인 전투 | 7.1% |
| 확신 | 6.5% |

→ 오전은 2곡 구조: **경배로 예배를 열고(경배·찬양 계열) + 그 주 정서의 응답곡**(인도·확신·사랑 등). 다만 **경배 오프너는 필수 아님**(왕상17엔 없이 인도+전투로 채움). 인도·돌보심이 최다인 건, 강해 본문 다수가 "주께서 인도하시니 나 따르리"류 응답을 부르기 때문.

## 2. 본문 '책'은 정서를 약하게만 규정한다

책별로 상위 분류를 뽑아보면 **거의 모든 책에서 헌신·인도가 상위** — 이는 위 **포지션 기본값의 반영**이지, 책 고유의 색이 아니다.

| 본문 책 | 주일수 | 성격 |
|---|---|---|
| **로마서** | **59** | 2024-08~2026-04 연속 강해(1~16장). 헌신·인도·확신·간증 고루 |
| 요한복음 | 15 | 인도·헌신·간증·경배 |
| 고린도전서 | 12 | 헌신·인도·사랑·성령(13장 사랑장 연작) |
| 마태복음 | 7 | 경배·인도·열망·헌신 |
| 고린도후서·열왕기상 | 각 6 | (왕상=2026 엘리야 시리즈) |
| 사도행전·다니엘·마가·계시록 | 각 4~5 | 초대·헌신·인도 |

**함의:** "무슨 책이냐"로 곡을 고르지 말 것. **그 주 본문이 성도에게 요구하는 '적용 정서'**를 먼저 정하고, **포지션 정서 지문(1절 표)** 안에서 곡을 맞춘다.

## 4. 추천 알고리즘 (이 분석의 최종 산물)

```
본문 받으면 →
 0) CSV(정답지)에 같은/인접 본문 실적 있으면 그게 1순위.
 1) 적용 정서 판별: "이 본문이 성도에게 요구하는 1인칭 응답이 뭔가?"
    (예: 확신 / 성결 / 헌신 / 열망 / 인도받음 / 위로 / 감사)
 2) 포지션 정서 지문에 맞춰 곡 배치:
    - 오전: (선택)경배 오프너 + 그 주 정서 응답곡  ※인도·돌보심 후보 풍부
    - 헌금: 179 기본 → 그 주 정서로 교체(성령·기도·간증·평안 계열)
    - 마무리: 헌신 풀(399·402·403·396·401)에서 그 주 색채로
 3) 2~3곡을 하나의 정서 아크로 통일. 곡은 '성도의 응답' 1인칭 위주.
```

> 참조 파일: 전수 대조표 `본문_찬송_enriched.md`, 빈도·포지션 요약 `선곡패턴_요약.md`.
